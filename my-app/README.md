# 🛒 Add to Cart React App (Fake Store API)

## 📌 Project Overview

This is a simple **ReactJS Add to Cart application** built using the Fake Store API.
Users can view products, add them to the cart, and manage cart items using a modal interface.

---

## 🚀 Features

* 📦 Fetch products from Fake Store API
* 🖼️ Display product image, title, and price
* 🛒 Add products to cart
* ⚠️ Prevent duplicate items with alert
* 🔢 Dynamic cart count in Navbar
* 🪟 Cart Modal to view added products
* ❌ Remove items from cart
* 📱 Fully responsive design
* 🎨 Styled using Tailwind CSS

---

## 🛠️ Tech Stack

* ReactJS (with Vite)
* JavaScript
* Tailwind CSS
* Axios (for API calls)

---

## 🌐 API Used

Fake Store API:
https://fakestoreapi.com/products

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone <your-repo-link>
```

2. Navigate to project folder:

```bash
cd my-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```

5. Open in browser:

```bash
http://localhost:5173/
```
